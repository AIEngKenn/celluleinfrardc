# HTTrack to Sanity Migration Output

Generated by:

```bash
node scripts/migrate-httrack-to-sanity.mjs
```

## Files

- `sanity-import.ndjson` contains Sanity-shaped documents for provinces, news categories, news, projects, procurement, and publications.
- `asset-manifest.json` lists every PDF/image reference found in the old HTML, with the source URL, expected local path, and suggested Sanity asset id.
- `migration-summary.json` contains document counts, asset counts, and migration notes.

## Important Notes

- The old site content is French. English fields currently duplicate the French text and should be translated later.
- Asset fields reference deterministic ids from `asset-manifest.json`. Upload those assets to Sanity first, or remap references after upload.
- Project budgets, progress, and geolocation are set to conservative defaults because the HTTrack HTML does not expose those values consistently.
